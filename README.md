- 此项目首先用在USFdown下载站中，后因非我方违背原则原因终止服务，此程序现成为独立项目。（目前合作已经恢复）
- ⚠️：此仓库版本为基础静态版本，我们正在努力编写动态高级版本，此版本只能通过修改JavaScript文件达到上架下架文件目的！
## 这是啥？
- 这是一个 静态动态混合架构 的文件分发程序，皆在提高站长的文件分发体验

## 特点一览
- 轻盈包体
- 简洁现代易上手的UI和UX
- 提供纯静态版本，强兼各类网站环境，低投入高回报
- 前端原生开发，无外部前端框架依赖（例如Vue)
- 支持高并发需求
